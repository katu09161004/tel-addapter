## 接続手順

### 1. DRA-H44 の接続

1. 電話機本体から受話器のカールコードを外す
2. DRA-H44 の「電話機側」端子を電話機本体に接続
3. DRA-H44 の「受話器側」端子に受話器のカールコードを接続
4. DRA-H44 の 3.5mm 出力をUSBオーディオアダプターのマイク入力へ

### 2. Raspberry Pi の接続

1. USBオーディオアダプターをRaspberry PiのUSBポートに接続
2. 電源を入れる

### 3. デバイス認識確認

```bash
# USB機器確認
lsusb

# オーディオデバイス確認
arecord -l
```

### 4. テスト録音

```bash
# カード番号を確認して実行（例: カード1, デバイス0）
arecord -D plughw:1,0 -f cd -t wav -d 5 test.wav

# 再生確認
aplay test.wav
```

## 注意事項

1. **対応電話機の確認**: 受話器が4極4芯カールコードで接続されていること
2. **側音の確認**: 自分の声が受話器で聞こえる電話機であること
3. **通話品質**: 相手の声・自分の声両方が録音される
4. **法的注意**: 通話録音は相手に告知することを推奨（ISO 15189対応として）
