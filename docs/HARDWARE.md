## 必須機材

### 1. テレホンピックアップ（電話カップラー）

**推奨: OM SYSTEM TP8**

| 項目 | 内容 |
|------|------|
| メーカー | OM SYSTEM (旧オリンパス) |
| 型番 | TP8 |
| 価格 | ¥1,500〜2,000 |
| 端子 | 3.5mm ステレオミニ |
| コード長 | 1.5m |
| 電源 | 不要（プラグインパワー） |

**購入先:**
- [Amazon](https://www.amazon.co.jp/dp/B007OXMHDE)
- [ヨドバシカメラ](https://www.yodobashi.com/product/100000001001494771/)
- [ヤマダ電機](https://www.yamada-denkiweb.com/228725013/)

**使用方法:**
1. イヤホン部分を耳に入れる
2. その上から電話の受話器を当てる
3. 通話音声がマイクに入力される

## 接続確認手順

### 1. ハードウェア接続

```
TP8 → USBオーディオ(マイク端子) → Raspberry Pi(USBポート)
```

### 2. デバイス認識確認

```bash
# USB機器確認
lsusb

# オーディオデバイス確認
arecord -l
```

### 3. テスト録音

```bash
# カード番号を確認して実行（例: カード1, デバイス0）
arecord -D plughw:1,0 -f cd -t wav -d 5 test.wav

# 再生確認
aplay test.wav
```

---

## 注意事項

1. **電話機への影響なし**: TP8は電話機に接続せず、耳に当てるだけ
2. **通話品質**: 相手の声・自分の声両方が録音される
3. **法的注意**: 通話録音は相手に告知することを推奨
